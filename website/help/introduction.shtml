Introduction

What is Wrestling Nerd?

Wrestling Nerd is computer software that assists in the management of wrestling tournaments. A user enters information like wrestler names, team names, and match results on a bracket on the screen. The software, in turn, automatically calculates team scores; determines fast fall leaders; prints bouts, brackets, scores, and place winners; exports results to common formats; and more. In effect, Wrestling Nerd tries to reduce some of the burden a person must shoulder to successfully run a large tournament.

Getting started

Removing old versions

Old versions of Wrestling Nerd should be uninstalled before a new version is installed. To uninstall Wrestling Nerd for Windows, go to the Control Panel and select Add/Remove Programs. Find and uninstall Wrestling Nerd in the list of installed programs. On other platforms, simply delete the Wrestling Nerd folder.

Installing on Windows 95, 98, ME, NT, 2000, XP

Users with any version of the Windows operating system may download a pre-built installer from the Wrestling Nerd home page. Running the installer automatically sets up Wrestling Nerd on the computer and places shortcuts to the program in the Start Menu and on the Desktop.

Installing on other platforms

Wrestling Nerd may also be run on any other platform supported by Python 2.2+ and the wxPython 2.5.3.1. Users may check out the Wrestling Nerd Python source code from the CVS repository on SourceForge. See the Wrestling Nerd home page for a link to the repository.
